# Mi Primer Repo

Mi primer repositorio en GitHub, con una calculadora web como proyecto inicial.

## Proyecto

**Calculadora web** — Una calculadora simple hecha con HTML, CSS y JavaScript.

### Funcionalidades

- Operaciones basicas: suma, resta, multiplicacion y division
- Porcentaje
- Soporte para teclado (numeros, operadores, Enter, Backspace, Escape)
- Borrar ultimo caracter y limpiar todo
- Historial de la ultima operacion

### Como usarla

Abre el archivo `index.html` en cualquier navegador (doble clic).

## Tecnologias

- HTML5
- CSS3
- JavaScript
